# California House Price Predictor 🏠🤖

## Project Overview
This project is a Machine Learning application designed to predict house prices in California. I built this to demonstrate the full data science lifecycle: from data cleaning to model optimization.

## Key Features
* **Exploratory Data Analysis (EDA):** Visualized correlations using Heatmaps to identify key drivers like Median Income.
* **Data Cleaning:** Identified and removed outliers (e.g., impossible occupancy rates) to ensure data quality.
* **Model Development:**
    * **Simple Linear Regression:** Baseline model.
    * **Multiple Linear Regression:** Improved performance by incorporating location and housing characteristics.
* **Performance Improvement:** Successfully reduced the **Mean Squared Error (MSE) from 0.69 to 0.46**.

## Tech Stack
* **Python**
* **Scikit-Learn** (Modeling)
* **Pandas & NumPy** (Data Handling)
* **Matplotlib & Seaborn** (Visualization)
